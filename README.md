# anton-dashboards

Data visualizations by [Sidney Rabsatt](https://github.com/rabsatt), built with [Anton](https://github.com/mindsdb/mindsdb).

## Dashboards

| Dashboard | Description | Sources |
|---|---|---|
| [⚡ The AI Skills Gap Clock](dashboards/ai_skills_gap_clock.html) | 342 occupations × 143M jobs scored for AI exposure | BLS |
| [💸 The Real Cost of a War](dashboards/real_cost_of_war.html) | Six US conflicts measured in dollars, debt, and lives | FRED · Watson Institute · DoD · 1941–2023 |
| [🏙️ When to Leave a City](dashboards/when_to_leave_a_city.html) | Rent trends and purchasing power by occupation and metro | Zillow · Census ACS · BLS · 2015–2026 |
| [📈 Who Actually Controls Inflation?](dashboards/who_controls_inflation.html) | Post-COVID inflation decomposed — what the data actually shows | FRED · BLS · 2000–2026 |
| [🗺️ How Old Are My Neighbors?](dashboards/neighbors.html) | Age and lifestyle by ZIP code across 31,433 US neighborhoods | Census ACS 2022 |

## Setup

Plain HTML/JS — no build step. Works with GitHub Pages or Vercel out of the box.

```bash
# GitHub Pages: enable in Settings → Pages → Deploy from branch → main / root
# Vercel: import repo, framework = Other, output dir = /
```
